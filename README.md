🚚 Dashboard de Gestão Logística e Performance Financeira (V2)

📌 Descrição do Projeto
Este projeto foi desenvolvido como uma solução de Business Intelligence para o setor logístico, focando no monitoramento de custos operacionais e rentabilidade. O dashboard transforma dados brutos de notas fiscais em indicadores estratégicos, permitindo que gestores identifiquem gargalos financeiros e desvios de metas em tempo real.

🛠️ Problema de Negócio Resolvido
O desafio era centralizar o acompanhamento de fretes e custos operacionais que, muitas vezes, ficam dispersos. A solução foca em responder:

Qual a nossa margem de lucro real por operação?

Quais cidades estão apresentando custos acima da meta estabelecida?

Quais notas fiscais precisam de auditoria por excesso de custo?

🚀 Diferenciais Técnicos (O que eu apliquei)
Modelagem de Dados & DAX: Criação de medidas avançadas para cálculo de Margem de Lucro (%), Status de Custo Condicional e KPIs de Performance.

UX: Implementação de Tooltips Dinâmicos (Dicas de Ferramenta) com gráficos de rosca para detalhamento de custos sem sobrecarregar o visual principal.

Target: Inclusão de linhas de referência (benchmarks) para identificar oscilações de custos fora do padrão (R$ 1.500,00).

Design Minimalista (Dark Mode): Interface desenvolvida com foco em legibilidade, simetria e navegação intuitiva.

📊 Ferramentas Utilizadas
Microsoft Excel: Estruturação da base de dados do zero. Apliquei conceitos de limpeza de dados, padronização de registros (Cidades, NFs, Clientes) e criação de massa de dados para simulação de cenários reais de logística.

Power BI: ETL (Extração, Transformação e Carregamento) via Power Query, modelagem de dados e criação de visualizações dinâmicas.

DAX: Desenvolvimento de fórmulas inteligentes para indicadores de performance (KPIs) e cálculos de rentabilidade.
